# TCESP_URs

Painel territorial em HTML para visualização das áreas fiscalizadoras do Tribunal de Contas do Estado de São Paulo, combinando DF/UR, PIB municipal, população, IEG-M e filtros dinâmicos por área fiscalizadora.

## Conteúdo

- `index.html`: aplicação principal em HTML único.
- Mapa com OpenStreetMap.
- Filtro por área fiscalizadora DF/UR.
- Indicadores: PIB, população, PIB per capita e IEG-M ponderado.
- Pins das sedes das Unidades Regionais.
- Links oficiais do TCESP e IBGE.

## Fontes utilizadas

- Tribunal de Contas do Estado de São Paulo — áreas fiscalizadoras, endereços, órgãos fiscalizados, IEG-M e portais institucionais.
- IBGE — PIB dos Municípios e estimativas populacionais.
